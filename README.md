# Test task Aspirity

## Installation

Using npm (port 5173)

```bash
  git clone https://github.com/NikSerdu/aspirity-test-task
  cd aspirity-test-task
  npm install
  npm run dev
```

Using docker compose (port 80)

```bash
  git clone https://github.com/NikSerdu/aspirity-test-task
  cd aspirity-test-task
  docker compose up -d
```
